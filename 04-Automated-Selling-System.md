# 4. Automated Selling System – Battleship – Midlife Fitness Reset

**Overview**  
The Battleship selling system automates the path from cold awareness → intake → diagnosis → enrolment → ongoing delivery.  
Goal: Scale to 10–30 active clients with minimal manual work after initial setup, using simple, resilient automations that mirror the programme itself.

**Core Funnel Stages & Automations**

1. **Awareness / Lead Generation**  
   - Organic: X/Twitter threads, short videos (Zone 2 walks, strength demos), content on "why past systems failed".  
   - Paid: Meta Ads (Facebook + Instagram) – primary channel for men 45–60.  
     **Facebook vs Instagram recommendation (2025–2026 data)**:  
     - **Facebook** is stronger for this audience: higher daily usage among 45–65 men, excellent for lead forms, retargeting warm audiences, private Groups, and interest targeting (men's health, weight loss over 40, hypertension, longevity). Cost per lead often lower; performs well with longer copy, testimonials, and "system mismatch" messaging.  
     - **Instagram** is secondary: better for visual proof (before/afters, transformation reels, walking videos) but lower engagement from men 50+ compared to Facebook. Use for brand awareness and retargeting lookalikes.  
     - Run both via Meta Ads Manager: unified targeting (age 45–65, men, interests like "fitness", "men's health", "blood pressure", behaviours like "engaged shoppers"). Start with £10–20/day tests, optimise for leads (intake quiz completions).  
   - Other: X posts, Reddit communities (r/fitnessover40, r/menshealth), YouTube shorts.

2. **Capture & Intake**  
   - Landing page (Carrd, Leadpages, or Pulse template): headline "Battleship – Midlife Fitness Reset: Simple system for men 45–60 who've failed before."  
   - Long-form intake quiz/form: past attempts, schedule, injuries, habits (incl. alcohol), goals.  
   - Auto-trigger Intake Agent → generates personalised Battleship Diagnosis Report ("Why failed + what’s different").  
   - Thank-you page: housekeeping video + join private FB Group + next steps.

3. **Nurture & Enrolment**  
   - Email automation (Pulse/Mailchimp):  
     - Day 0: Welcome + diagnosis report  
     - Day 1–3: Value emails (Zone 2 explanation, alcohol impact, mindset reframes)  
     - Day 4–7: Case study / testimonial + enrolment call booking  
   - Private FB Group: pinned Battleship welcome post, daily tips, community support.  
   - Sales call / application: low-pressure, system-focused pitch. Payment via Stripe link.

4. **Onboarding & Delivery**  
   - Payment confirmed → Program Agent generates full 12-week Battleship Plan (PDF/MD in client portal or email).  
   - Onboarding sequence: setup MyFitnessPal, tracker, logging instructions.  
   - Group access + weekly check-in automation starts.

5. **Ongoing Retention & Upsell**  
   - Weekly Check-in Agent: analyses logs → adjustments + personalised coach message ("Great steps compliance! Add 10 min post-dinner walk this week.").  
   - Education Agent drips lessons (why diets fail, protein for longevity, realistic alcohol guidance).  
   - Monthly membership upsell after 12 weeks: continued check-ins, plan refreshes, community.

**Tools & Integrations**  
- **CRM / Automation**: Pulse (or ActiveCampaign/Zapier) for emails, pipelines, forms.  
- **Payments**: Stripe (links in emails/landing page).  
- **Tracking**: MyFitnessPal (client-side food/steps).  
- **Ads**: Meta Ads Manager (Facebook primary, Instagram secondary).  
- **Agents**: OpenClaw skills for intake/program/check-in/education routing.  
- **Community**: Private Facebook Group (easy for 45–60 demographic).  
- **Optional**: Postiz for automated social posting.

**Key Battleship Automations to Build (via OpenClaw/Claude)**  
- Form submission → Intake Agent → diagnosis MD → email delivery  
- Weekly logs upload → Check-in Agent → adjustment + message  
- Progress milestone → Education Agent → relevant lesson drip  
- Ad retargeting audiences: quiz completers who didn’t enrol → nurture sequence

**Ad Budget Guidance**  
- Start small: £10–30/day on Facebook lead ads (intake quiz objective).  
- Scale when CPL < £10–15 and conversion to paid > 10–15%.  
- Use lookalikes from past clients + interests (men's fitness, weight loss 40+, health trackers).
