### 2. 11-Week Bodyweight Full-Body Program

**Frequency:** 3× per week  
**Focus:** Progressive calisthenics for strength & muscle using bodyweight only  
**Equipment:** None required (optional: pull-up bar, sturdy table/doorway for rows)

**Exercises (perform in this order each session):**
- Push-Up variation
- Squat variation (Air Squat → Pistol progression)
- Inverted Row / Superman Hold
- Lunge variation (Walking or Jump Lunges)
- Pike Push-Up / Shoulder Tap progression
- Plank variation (standard → side → hollow)

**Progression Blocks:**
- Weeks 1–3: Build volume with easier variations
- Weeks 4–6: Add sets + intermediate variations
- Weeks 7–9: Harder variations + lower reps/higher intensity
- Weeks 10–11: Advanced variations + pauses/slow tempos for max stimulus
