### 11-Week Home Complete Program (Dumbbells + Bands + Pull-Up Bar)

**Frequency:** 3× per week — alternating Session A and Session B
**Focus:** Full-body strength with supersets — maximum output from home equipment
**Equipment:** Dumbbells (adjustable ideal), resistance bands, pull-up bar

This is a proper strength programme. The superset structure means each session is 35–40 minutes but the intensity is high — you're working two muscle groups simultaneously with 60 seconds rest between pairs. This is how you build muscle and burn fat at the same time without spending two hours training.

**Session A — Push + Legs:**
- SS1: Goblet Squat + Push-Up (or DB Floor Press)
- SS2: Dumbbell Shoulder Press + Reverse Lunge (per leg)
- SS3: Lateral Raise + Wall Sit or Plank

**Session B — Pull + Hinge:**
- SS1: Dumbbell Row + Romanian Deadlift (RDL)
- SS2: Pull-Up (or Band Pulldown) + Dumbbell Bicep Curl
- SS3: Band Face Pull + Dead Bug or Bird Dog

**How supersets work:** Do exercise 1, rest 5–10s, do exercise 2, then rest 60s. That's one set. Repeat for the number of sets that week. This keeps your heart rate up and cuts session time in half.

**Push-Up Challenge (daily — separate from sessions):**
3 × max reps every day. Record the number. This builds on top of the programme — it's not part of the sessions.

**Progression Blocks:**
- Weeks 1–3: 3 sets per superset — learn the pairing rhythm, moderate weight
- Weeks 4–6: 3–4 sets — increase load, tighten rest to 60s
- Weeks 7–9: 4 sets — heavier dumbbells, push near failure on last set
- Weeks 10–11: 4 sets + intensity technique on final superset (slow eccentric, rest-pause)
