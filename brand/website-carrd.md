## DNS — POINTING GODADDY → CARRD

In Carrd: Sites → your site → Share → Custom Domain → type battleshipreset.com → Carrd will show you the exact records needed.

**Typical Carrd DNS records (confirm in your Carrd dashboard):**

| Type  | Host | Value           | TTL  |
|-------|------|-----------------|------|
| A     | @    | 76.76.21.21     | 600  |
| CNAME | www  | alias.carrd.co  | 600  |

**In GoDaddy:**
1. Domain Portfolio → battleshipreset.com → DNS
2. Delete any existing A record for @ (the root domain)
3. Add the A record above
4. Add/edit the CNAME for www
5. Save — propagation takes 15 min to a few hours

**Carrd plan needed:** Pro Lite (£9/year) minimum for custom domain.
