# Master GBP Dashboard Prompt

**Use this when someone wants a "full GBP audit," "complete optimization report," or "master dashboard" in one go.**

This prompt orchestrates multiple audits into a single, prioritized output.

## The Master Prompt (copy-paste ready — user fills in [placeholders])

"Act as my local SEO dashboard builder.  
I want a comprehensive Google Business Profile optimization report and action plan.  

My GBP: [my GBP URL]  
Competitors: [URL1], [URL2], [URL3]  
My core services: [service1], [service2], [service3]  
My main service areas: [area1], [area2], [area3]  
Target keywords: [keyword1], [keyword2], [keyword3]  

Run the following audits in parallel where possible, then compile everything into one master spreadsheet/report with tabs/sections:  

1. Category Audit (primary + secondaries — recommend ideal set)  
2. Attributes Audit (highlight missing ones, baseline shared by all competitors)  
3. Review Teardown (velocity comparison, monthly target to catch leader, top mentioned services/neighborhoods/complaints)  
4. Review Response Analysis (my vs competitors — response rate, time, quality; provide template system with variations)  
5. Posts Strategy (competitor posting frequency/topics; build 8-week calendar with first 4 weeks written)  
6. Services Section (compare vs competitors + website [my website URL]; write optimized 2-3 sentence descriptions)  
7. Description Optimization (analyze competitors; write 3 versions: keyword-heavy, conversion-heavy, balanced)  
8. Photo Audit (count/types/recency; provide 8-week upload plan focused on before-afters, team, trucks in areas)

Output structure:  
- Executive Summary: Top 5 highest-impact fixes ranked by potential speed/ROI (e.g., change primary category first).  
- Master Spreadsheet (describe tabs): one tab per audit + a 'Prioritized Action Plan' tab with columns: Task, From Which Audit, Estimated Impact (High/Med/Low), Time to Implement, Expected Outcome.  
- Quick Wins (things I can do today/this week).  
- 90-Day Roadmap: what to focus on each month.  

Prioritize based on 2026 local SEO factors: relevance (categories first), prominence (reviews/photos/activity), then ongoing signals."

## Why This Matters

Users love one-click "do everything" requests. This master prompt turns your bot into a full GBP strategist — it chains the individual audits, avoids redundancy, and delivers a clean, actionable dashboard instead of scattered reports.
